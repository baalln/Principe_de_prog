Ce projet est un mini-projet qui sert à comprendre comment utiliser le soap.
Pour l'utilisation de ce projet, nous avons utilisé IntelliJ et SoapUI.

Dans ce projet, nous avons créé 3 classes MonserviceWeb, Application et Étudiant.

Grâce à la classe Appilctaion on peut lancer un service implémenté dans la classe MonserviceWeb.

SopaUI va nous permmetre d'interagir avec les services créé.


L'utiliastion du projet:

-On lance ala classe application.
-on ouvre un onglet "http://localhost:8888/?wsdl".
-on ouvre SoapUI.
